Foodie Q&A

🤔 Foodie 미식회, 그게 뭔가요?

🚕 교통비는 지원되나요?

👨‍🏫 네! 미식회 장소까지 이동 택시비를 전액 지원합니다.

🍻 끝나면 바로 헤어지나요?

👨‍🏫 1차 후엔 모든 조가 함께 모여 친목 타임! 감동을 나누고 친구를 사귀어요.

Foodie Web Service는 뭔가요?

🧭 왜 만들었나요?

  • 예약/대기 자동화로 공정성 확보
  • 실시간 진행, 보관 기록, 리뷰 관리까지 한 곳에서
  • 조별 점수·활동 로드맵을 모두가 쉽게 확인

🛠 어떻게 쓰나요?

  1. 학번/이름을 입력해 입장합니다.
  2. 탑승 게이트에서 식당을 선택하면 예약(정원 초과 시 자동 대기)됩니다.
  3. 예약 취소는 카드의 탑승 취소 버튼으로 간단히.
  4. 아이디어 건의 탭에서 익명 제안 가능.
  5. 관리자는 관리자 대시보드/임원 관리 탭으로 운영합니다.
