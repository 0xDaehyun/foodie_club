🍜 푸디 미식회 예약 시스템
맛집탐방 동아리 '푸디'를 위한 실시간 미식회 예약 웹 애플리케이션입니다.

이 프로젝트는 동아리원들이 모바일이나 PC를 통해 간편하게 미식회 참여 예약을 하고, 실시간으로 예약 현황을 확인할 수 있도록 돕기 위해 만들어졌습니다.

✨ 주요 기능
실시간 예약 시스템: 선착순으로 예약을 받고, 정원이 차면 자동으로 대기자 명단에 등록됩니다.

실시간 현황 공유: 한 명이 예약하거나 취소하면, 접속해 있는 모든 사람의 화면이 동시에 업데이트됩니다.

관리자 기능: 동아리 회장은 새로운 미식회(이벤트)를 생성하고 참여 인원을 설정할 수 있습니다.

식당 정보 제공: 관리자가 미식회 장소 정보를 등록하면, 모든 사용자가 예약 페이지에서 확인할 수 있습니다.

간편한 본인인증: 별도의 회원가입 없이 학번과 이름만으로 본인을 식별하여 예약에 참여합니다.

🚀 기술 스택
Frontend: HTML, Tailwind CSS, JavaScript

Realtime Database: Google Firebase (Firestore)

Deployment: GitHub Pages

🔧 배포 방법
이 저장소에 있는 index.html 파일이 웹페이지의 본체입니다.

GitHub 저장소의 Settings > Pages 메뉴로 이동합니다.

배포 소스(Source)를 Deploy from a branch로 설정합니다.

브랜치(Branch)를 main (또는 master)으로 선택하고, 폴더는 /root로 둔 뒤 Save를 누릅니다.

잠시 후 https://{사용자이름}.github.io/{저장소이름}/ 주소로 웹페이지가 배포됩니다.

참고: 이 프로젝트는 Firebase와 연동이 필요합니다. 직접 배포하여 사용하려면, index.html 파일 내의 Firebase 설정 코드를 본인의 Firebase 프로젝트 설정 값으로 변경해야 합니다.
